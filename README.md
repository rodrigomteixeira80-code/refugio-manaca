# Refúgio Manacá

Site institucional da pousada Refúgio Manacá, localizada em Cunha - SP.

## Objetivo

Criar uma página simples, moderna e responsiva para apresentar a pousada, suas acomodações, contato e informações para reservas.

## Tecnologias usadas

- HTML
- CSS
- JavaScript
